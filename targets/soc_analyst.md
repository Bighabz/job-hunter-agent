# Target Archetype: SOC Analyst / Cybersecurity Analyst

## Priority Keywords (front-load in resume)
- Security Operations Center (SOC)
- Incident response / incident handling
- SIEM (Splunk, QRadar, Sentinel — specify if in JD)
- Threat detection / threat hunting
- Log analysis / log correlation
- Vulnerability assessment / vulnerability management
- IDS/IPS (Snort, Suricata)
- Endpoint detection and response (EDR)
- Network traffic analysis
- Malware analysis (basic)
- Ticketing systems (ServiceNow, Jira)
- MITRE ATT&CK framework
- Kill chain methodology
- Phishing investigation
- Packet capture / Wireshark

## Certifications to Emphasize
1. CompTIA Security+ (primary — most JDs require this)
2. ISC2 CC (shows commitment to ISC2 track → CISSP path)
3. CompTIA A+ (foundations, especially for Tier 1)

## Experience to Front-Load
- El Camino College SOC experience (direct match)
- ClaWD red team work (shows attacker perspective — valuable for blue team)
- AI Horizon DCWF analysis (shows understanding of cybersecurity workforce landscape)
- Phishing origin story (real-world threat experience)

## Experience to De-Emphasize
- Armed security (mention briefly for clearance/professionalism angle)
- Crypto/DeFi (unless Web3 security role)

## Section Order (Resume)
1. Summary / Professional Profile
2. Certifications
3. Technical Skills
4. SOC / Security Experience
5. Projects (ClaWD, AI Horizon)
6. Education

## Cover Letter Angle
- "Attacker mindset informs my defense" — built autonomous red team agent, now applying that understanding to detection
- Real phishing experience → genuine passion for protecting organizations
- AI + security intersection — automation skills for SOC efficiency (SOAR-adjacent)
